About the Package

Vedic Calendar System for R

VedicDateTime implements the traditional Vedic calendar system used in the Indian subcontinent. The package provides functions to calculate all five elements of the Panchanga (five limbs of time), enabling users to determine dates and timings based on traditional astronomical calculations.

Built on the Swiss Ephemeris for precise astronomical computations, VedicDateTime is useful for researchers, developers, and anyone interested in Vedic timekeeping and calendar systems.

R Console
# Install from CRAN
install.packages("VedicDateTime")
 
# Load the package
library(VedicDateTime)
 
# Get Panchanga elements for a date
date <- as.Date("2024-01-15")
tithi(date)
#> [1] "Panchami"
nakshatra(date)
#> [1] "Rohini"
vaara(date)
#> [1] "Somvar"
yoga(date)
#> [1] "Saubhagya"
karana(date)
#> [1] "Taitila"
Capabilities

Key Features

Calculate all five elements of the traditional Panchanga calendar system

🌙

Tithi

Calculate the lunar day based on the angular distance between Sun and Moon. 30 tithis per lunar month.

⭐

Nakshatra

Determine which of the 27 lunar mansions the Moon currently occupies in the zodiac.

☀

Vaara

Get the Vedic weekday name with its associated planetary deity and characteristics.

☯

Yoga

Compute the 27 yogas formed by the combined longitudes of the Sun and Moon.

◐

Karana

Identify the half-tithi period. There are 11 karana types used in Vedic astrology.

📅

Masa and Ritu

Determine the lunar month and season classifications of the Vedic calendar.
